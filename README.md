# 🐶 Alimentador Automático para Pets com ESP32

Projeto desenvolvido para automatizar a alimentação de pets utilizando ESP32 com controle via aplicativo.

## 📌 Descrição
O sistema permite a liberação de ração de forma automática ou remota através de um aplicativo, proporcionando praticidade e controle ao usuário.

## ⚙️ Tecnologias Utilizadas
- ESP32
- Linguagem C/C++
- Integração com aplicativo

## 🚀 Funcionalidades
- Liberação automática de ração
- Controle remoto via aplicativo
- Sistema embarcado com comunicação entre hardware e software
